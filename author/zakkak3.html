How to create local git repositories

  1. Create the repo directory (the '.' prefix is to make it hidden)

    mkdir .myrepo.git

  2. Go inside the directory

    cd .myrepo.git

  3. Initialize the git repo

    git init --bare

Our repo is now ready

to clone:

git clone /path/to/.myrepo.git where_to_clone

Changing user id in linux

As a different user execute

usermod -u  username
find / -user ; -exec chown -h ; {} \;

example:

usermod -u 2179 user1
find / -user 1000 -exec chown -h 2179 {} \;

SSH login without password

Open your favorite terminal emulator and type

ssh-keygen -t dsa

(simply enter on whatever it asks you)

try

ssh-copy-id -i ~/.ssh/id_dsa.pub user@remote.machine.com

if it fails (you don't have ssh-copy-id) do

cat ~/.ssh/id_dsa.pub | ssh user@remote.machine.com 'cat >> ~/.ssh/authorized_keys'
