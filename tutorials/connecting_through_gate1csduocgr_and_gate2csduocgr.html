Connecting through gate1.csd.uoc.gr and gate2.csd.uoc.gr

Date Tue 01 June 2010

Απλό SSH:

  1. ssh gate1.csd.uoc.g​r

  2. Αφού εισάγετε των κωδικό σας και εμφανιστεί το %

    ssh milo

  3. θα σας τυπώσει το παρακάτω μήνυμα

    The authe​nticity of host 'milo.csd.uoc.gr (147.52.19.9)' can't be established. RSA key fingerprint is 5a:37:48:87:3a:d3:37:4e:34:15:4b:d2:3f:07:bc:aa. Are you sure you want to continue connecting (yes/no)?

    Γράφεται yes

  4. Εισάγεται τον κωδικό σας και είστε έτοιμοι

Απλό SSH με χρήση enable:

  1. ssh gate1.csd.uoc.gr

  2. Αφού εισάγετε των κωδικό σας και εμφανιστεί το %

    enable milo

  3. θα σας τυπώσει το παρακάτω μήνυμα

    Th​e authenticity of host 'milo.csd.uoc.gr (147.52.19.9)' can't be established. RSA key fingerprint is 5a:37:48:87:3a:d3:37:4e:34:15:4b:d2:3f:07:bc:aa. Are you sure you want to continue connecting (yes/no)?

     Γράφεται yes

  4. Εισάγεται τον κωδικό σας και θα εμφανιστεί ενα μήνυμα σαν αυτό

    You can now connect to milo.

1. Linux commands:

ssh zakkak@gate1.csd.uoc.gr -p 10705
scp -P 10705 <filename> zakkak@gate1.csd.uoc.gr:</full/dest/path>
sftp -o port=10705 zakkak@gate1.csd.uoc.gr

2. Windows settings (SSH Secure Shell Client or replace scp/sftp with pscp/psftp):

Username: zakkak
Hostname: gate1.csd.uoc.gr
Port: 10705

For detailed info, go to http://www.csd.uoc.gr/services/useful-info/remote-connection.html

  5. Από το παραπάνω μήνυμα κάνουμε αντιγραφή και επικόληση σε άλλο τερματικό (προσοχή το port θα διαφέρει κάθε φορά)

    ssh zakkak@gate1.csd.uoc.gr -p 17440

  6. θα σας τυπώσει το παρακάτω μήνυμα

    The authenticity of host 'gate1.csd.uoc.gr (147.52.19.9)' can't be established. RSA key fingerprint is 5a:37:48:87:3a:d3:37:4e:34:15:4b:d2:3f:07:bc:aa. Are you sure you want to continue connecting (yes/no)?

    Γράφεται yes

  7. Εισάγεται τον κωδικό σας και είστε έτοιμοι
